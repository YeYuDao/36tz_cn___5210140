<br/></br>### 课程介绍:
本课程系统介绍算法设计与分析的方法和理论，包括算法基础、图、贪婪算法、分治、动态规划、网络流、计算复杂性初步、近似算法及随机算法等。同时，本课程还包含算法领域的一些前沿课题和最新进展。本课程可以作为数学、计算机等相关专业的研究生和高年级本科生关于算法理论的基础课程。
